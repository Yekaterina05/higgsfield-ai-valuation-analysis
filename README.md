# Reverse-Engineering the Deal: Higgsfield AI Series A Extension

Author: Yekaterina Pyurko  
Course: TSIS 1  
Date: January 2026  

Project Overview

This project reconstructs the Series A Extension financing round of Higgsfield AI using a reverse-engineering valuation approach.

Investment: $80,000,000  
Post-Money Valuation: $1,300,000,000  
ARR Run Rate: $200,000,000  

The analysis derives:
- Pre-money valuation
- Investor ownership
- Dilution impact
- Cap table evolution

Financial Reconstruction

Pre-Money Valuation

Vpre = Vpost − Investment  
Vpre = 1,300,000,000 − 80,000,000  

Result: $1,220,000,000

Investor Stake

Stake = Investment / Post-money  

80,000,000 / 1,300,000,000 ≈ 6.15%

Dilution

Dilution ≈ 6.15%

Cap Table (Post-Money)

Total Shares: 1,065,574

Founders – 75.08%  
Option Pool – 18.77%  
Series A Extension Investors – 6.15%

Investor Breakdown:
- Accel: 3.50%
- Menlo Ventures: 1.50%
- GFT Ventures: 1.15%

Revenue Multiple

$1.3B valuation at $200M ARR  
≈ 6.5× revenue multiple
